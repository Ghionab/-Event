# Quick Start: Testing Check-in System

## 1. Start the Server (30 seconds)

```bash
python manage.py runserver 8002 --settings=event_project.settings_staff
```

## 2. Generate QR Codes (10 seconds)

```bash
python generate_test_qr.py
```

This creates PNG files like `qr_REG-E560E8D4.png` in your project folder.

## 3. Login (30 seconds)

1. Open browser: http://localhost:8002/staff/login/
2. Login with: `staff@test.com` (use your staff password)

## 4. Test QR Scanning (1 minute)

1. Click on any event from the list
2. Click "Start Scanning" button
3. Open one of the generated QR code PNG files on your phone
4. Point camera at the QR code
5. ✅ Should see: "✓ [Name] checked in successfully!"

## 5. Test Manual Check-in (30 seconds)

1. Find an attendee with "CONFIRMED" status
2. Click the "Check In" button
3. Click "OK" in confirmation dialog
4. ✅ Should see success message and green background

## 6. Verify (30 seconds)

```bash
python verify_checkin_setup.py
```

Should show: `✅ ALL CHECKS PASSED!`

---

## Troubleshooting

### Camera won't start?
- Allow camera permissions in browser
- Try Chrome or Edge browser

### QR code won't scan?
- Ensure good lighting
- Move camera closer/farther
- Make sure QR code is clear and large enough

### "Invalid QR code" error?
- Make sure you're scanning a QR code generated by `generate_test_qr.py`
- Check that the registration is CONFIRMED status

### Check-in button doesn't work?
- Make sure you're logged in as staff
- Check browser console (F12) for errors
- Verify registration status is CONFIRMED
